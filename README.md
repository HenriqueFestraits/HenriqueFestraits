# Featured Projects

## K'iin

A 2D Top-Down RPG developed in Java and built on top of a custom game engine. The project applies software engineering principles such as modular architecture, documentation and separation between reusable infrastructure and game-specific logic.

---

## Engine 2D

A reusable Java game engine designed to support multiple 2D games through independent modules responsible for rendering, resources, entities, physics, animation, camera, audio, UI and state management.

---

## Virtual LaunchPad

Desktop application developed with Java Swing and MySQL through JDBC, featuring user authentication, preset management and a simple polyphony system capable of reproducing multiple instances of the same sound simultaneously.

---

## Calculadora de Reclusão

Android application developed in Kotlin for a real client, implementing prison sentence progression calculations based on Brazilian legislation and integrating WhatsApp for lawyer communication.
